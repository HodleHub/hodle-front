# Como começar na Hodle  
### Um guia simples e direto para comprar Bitcoin usando DEPIX

A **Hodle** é uma plataforma brasileira que permite comprar Bitcoin de forma simples, rápida e com **low-KYC**. ou seja, **sem necessidade de enviar documentos**. Toda a verificação é feita automaticamente utilizando os **dados do próprio PIX**, garantindo segurança e privacidade.

Se você quer comprar Bitcoin com praticidade, segurança e sem burocracia, este guia é para você.

---

# Limites de compra na Hodle

A Hodle utiliza um sistema progressivo que libera limites maiores conforme você utiliza a plataforma:

| Nível | Requisito | Limite diário |
|------|-----------|----------------|
| **Nível 1** | Primeira compra | **R$ 100** |
| **Nível 2** | Após 5 compras | **R$ 500** |
| **Nível 3** | Após 10 compras | **R$ 6.000** |

# Por que a Hodle usa DEPIX?

Toda compra de Bitcoin na **Lightning**, **Liquid** ou **On-chain** é intermediada pela stablecoin **DEPIX**.

## O que é DEPIX?

DEPIX é uma stablecoin lastreada **1:1 no real**, emitida na **Liquid Network**.

O token tem o objetivo de ser a porta entrada entre o mundo fiat e a blockchain, um token transiente com o objetivo de ser trocado por BTC

Whitepaper oficial para mais detalhes:  
https://github.com/eulen-repo/DePix/blob/main/whitepaper/depix_whitepaper-pt_BR.pdf

### Vantagens:
- Liquid oferece **segurança e confidencialidade**  
- Transações rápidas e baratas  
- Token transitório: feito para uso imediato  
- Swap simples para BTC dentro da carteira

# Como comprar na Hodle (passo a passo)

### **1. Gere um endereço Liquid na sua carteira**

Exemplo de endereço válido:

lq1pqv32yn0k23333w55gd3qldsfsrr85ehcuesjudpg0uzjfavatd08lemwengmdtzuwvgf6cvcdvxvp7sqzaasatumrkqf0sraljhevcylpegzer9wvj7u

### **2. Acesse https://hodle.com.br/**  
Selecione o ativo **DEPIX**.

### **3. Informe o valor e cole seu endereço Liquid**

![Tutorial passo 3](/tutorial-1.png)

### **4. Pague o QR Code PIX**

![Tutorial passo 4](/tutorial-2.png)

Após o pagamento:
- Seus DEPIX são emitidos  
- São enviados automaticamente para sua carteira Liquid  

### **5. Faça o swap para Bitcoin**

Como o DEPIX é um token transitório, recomenda-se converter para Bitcoin logo após o recebimento.

---

# Como fazer swap para Bitcoin (exemplo usando SideSwap)

1. Abra o SideSwap  
2. Vá até **Swap**  
3. Escolha **DEPIX → L-BTC**  
4. Revise a cotação  
5. Toque em **Swap**

![Tutorial passo 5](/tutorial-3.png)

Em segundos seus Bitcoins estarão na carteira.
